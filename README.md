# prediction_geometry_figures

##Описание задачи
В этой задаче необходимо классифицировать геометрические фигуры на RGB-изображениях размером 64×64 пикселя.

Дано:

- Набор обучающих изображений с четырьмя классами фигур: круг, квадрат, треугольник и пятиугольник
- Каждое изображение содержит одну фигуру чёрного цвета на цветном фоне
- Обучающий набор содержит ~12000 изображений, тестовый набор содержит ~3000 изображений
Задача: Разработать алгоритм классификации, который для каждого изображения из тестового набора предсказывает класс фигуры.
